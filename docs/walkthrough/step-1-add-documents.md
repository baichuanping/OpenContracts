# Step 1: Add Documents

For detailed information on all the ways to get documents into OpenContracts,
see the [Document Upload Methods](../upload_methods/index.md) guide.

The quickest way to get started is the single document upload:

1. Navigate to the **Documents** tab

   ![](../assets/images/screenshots/Empty_Documents_View.png)

2. Click the **Action** dropdown and select **Import**

   ![](../assets/images/screenshots/Doc_Action_Menu.png)

3. Drag and drop files or click to browse

   ![](../assets/images/screenshots/Import_Documents_Modal.png)

4. Edit titles and descriptions if desired

   ![](../assets/images/screenshots/Edit_Document_Details.png)

5. Click **Upload**

OpenContracts accepts PDFs, Word documents (.docx), and plain text files. Once
uploaded, documents are automatically processed by the document parser (Docling
by default) to extract text, structure, and token positions. While processing,
the document shows a loading indicator and cannot be added to a corpus.

![](../assets/images/screenshots/Document_Processing.png)

For more details on the processing pipeline, see the
[Pipeline Overview](../pipelines/pipeline_overview.md).
